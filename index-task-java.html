Страница Задачник

Задачник по языку программирования — это сборник задач, предназначенных для практического применения и углубления знаний по определенному языку программирования. Он может быть полезным для начинающих программистов, студентов, а также опытных разработчиков, желающих улучшить свои навыки.

Задачники по языку программирования, позволяют разработчикам практиковаться в написании кода, решении конкретных задач и изучении особенностей языка. Эти задачи могут варьироваться по сложности и охватывать разные аспекты программирования, включая базовые конструкции языка, алгоритмы, структуры данных, объектно-ориентированное программирование и другие темы.

Напишите программу, которая выводит на экран “Привет, мир!”.

System.out.println("Привет, Мир!");

Напишите программу, которая запрашивает у пользователя два числа и выводит их сумму.

Scaner scaner = new Scaner(System.in);

System.out.print("Введите первое число: ");
int number1 = scaner.nextInt();

System.out.println("Введите второе число: ");
int number2 = scaner.nextInt();

int sum = number1+number2;

System.out.println("Сумма чисел: " + sum);

Напишите программу, которая запрашивает у пользователя радиус окружности и вычисляет ее площадь.

Scaner scaner = new Scaner(System.in);

System.out.print("Введите радиус окружности: ");
double radius = scaner.nextDouble();

double area = Math.PI * radius * radius;

System.out.println("Площадь окружности: " + area);

Напишите программу, которая запрашивает у пользователя строку и выводит ее длину.

Scaner scaner = new Scaner(System.in);

System.out.print("Введите строку: ");
String input = scaner.nextLine();

int length = input.length();

System.out.println("Длина строки: " + length);

Напишите программу, которая запрашивает у пользователя температуру в градусах Цельсия и выполняет её перевод в градусы Фаренгейта.

Scaner scaner = new Scaner(System.in);

System.out.print("Введите температуру в градусах Цельсии: ");
double celsius = scaner.nextDouble();

double fahrenheit = ( celsius * 9/ 5 ) + 32;

System.out.println("Температура в градусах Фаренгейта: " + fahrenheit);
