Переменные

Переменные в Python используются для хранения данных. Их можно рассматривать как именованные контейнеры, которые могут содержать разные типы данных, такие как числа, строки, списки и т. д. Переменные создаются путем присвоения им значения, например:

x = 5
name = "Джон"

В приведенном примере мы объявляем две переменные с именем "x" и "name". В переменной "x" присваиваем ей значение 5, а в переменной "name" присваиваем ей строковое значение Джон.

Типы данных

Типы данных определяют вид данных, которые может содержать переменная. Python имеет несколько встроенных типов данных, в том числе:

Числовые типы данных

  • int: Целочисленный тип данных. Например: 10, -5, 0.
  • float: Вещественный тип данных. Например: 3.14, -2.5, 0.0.
  • complex: Комплексный тип данных. Например: 2 + 3j, -1 + 4j.
my_integer = 10
my_float = 3.14
my_complex = 2+3j

Строковый тип данных:

Строки представляют последовательность символов и обозначаются с помощью одинарных ('') или двойных ("") кавычек.

my_string = "Hello, World!"

Логический тип данных:

Логический тип данных представляет булево значение True или False.

my_boolean = True

Списки (list):

Список - это упорядоченная коллекция элементов, которые могут быть разных типов данных. Элементы списка разделяются запятыми и заключаются в квадратные скобки ([])

my_list = [1, 2, 3, "four", 5.0]

Словари (dictionary):

Словарь - это неупорядоченная коллекция элементов, состоящая из пар ключ-значение. Ключи должны быть уникальными, и они используются для доступа к значениям. Словари обозначаются фигурными скобками ({}) и элементы разделяются запятыми.

my_dict = {
"name" : "John",
"age" : 25
}
